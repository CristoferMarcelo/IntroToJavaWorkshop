Where's Waldo

Steps:

  1. Open WheresWaldo.pde using Processing.
  2. Choose a puzzle image from the "img" folder, then drag and drop it into your processing window
  3. On line 2 of your sketch declare a variable for the image: 
    PImage puzzle;
  4. In your setup() method, import your image using the following code: 
    puzzle = loadImage(“puzzle.jpg”);
  5. Set the size of your sketch and the size of your image to be the same by entering the following code in the setup() method. 
    size(800,600);
puzzle.resize(width,height);
  6. Use the solutions within the "img/solutions" folder to locate Waldo in the puzzle. He is labeled with the number "1".
  7. You now need to imagine a square around Waldo, and click on the corners of your imaginary square while recording the x and y location of each corner. Use the code below to print out where you click. 
    if(mousePressed){
    println("Mouse’s x-position: " + mouseX + "\n" +
     "Mouse’s y-position: " + mouseY + "\n");
}
  8. To see if we clicked on Waldo, use this code and change the values to the numbers you found in step 7. 
    if(mousePressed){
   if(mouseX > minimumXValue && mouseX < maximumXValue){
        if(mouseY > minimumYValue && mouseY < maximumYValue){
            showMessageDialog(null, "You Found Waldo!");
        }
    }
}
  9. EXTRA: Repeat the steps above to alert users when they identify additional characters within the puzzle. Depending on the puzzle you selected, additional characters are labeled with the following numbers:
    • Woof the dog...'s tail (#2)
    • Wilma, Waldo's girlfriend (#3)
    • Whitebeard the Wizard (#4)
    • Odlaw (or anti-Waldo) (#5)
