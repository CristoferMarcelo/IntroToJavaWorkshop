Googly Eyes

Steps:

  1. Open GooglyEyes.pde using Processing.
  2. Choose a face image from the "img" folder, then drag and drop it into your processing window. Load it like this in the setup() method: 
    PImage face = loadImage(“face.jpg”);
  3. In your setup method, set the size of your sketch and the size of your image to be the same. 
    size(800,600);
face.resize(width,height);
  4. Place 2 white ellipses over the eyes.
  5. Now add 2 smaller black ellipses to the eyes to represent the pupils.
  6. Change your fill() command to use mouseX and mouseY to change the color of the irises when the mouse moves.
  7. Make sure you SAVE YOUR CODE when you are done.
